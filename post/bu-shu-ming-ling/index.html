部署命令

2019-11-09

cd /data/service 进入目录
mkdir tl-gasoline
cd tl-gasoline
上传jar包
nohup java -jar tl-gasoline-0.0.1-SNAPSHOT.jar --spring.profiles.active=prod >> catalina.out 2>&1 &
部署项目
tail -f -n 200 catalina.out
查看是否成功
配置nginx

service nginx restart

ps -ef | grep java
